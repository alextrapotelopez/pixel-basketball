# Pixel Court

A browser-based pixel basketball game with selectable NBA teams, real player names and jersey numbers, live stats, overtime, jump ball, steals, blocks, dunks, and 5v5 play.

## Play

Open `index.html` in a browser.

## Controls

- `WASD` or arrow keys: Move
- `Space`: Jump ball / hold and release to shoot
- `C`: Dunk near the hoop
- `E`: Pass
- `Q`: Steal on defense
- `F`: Block on defense
- `Shift`: Sprint
- `R`: Restart
