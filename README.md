# Pixel Court

A browser-based pixel basketball game with selectable NBA teams, real player names and jersey numbers, live stats, four 2-minute quarters, 30-second overtime periods, jump ball, steals, blocks, dunks, 5v5 play, and a serverless NBA-style league mode.

League Mode saves one local season in the browser with East/West conferences, an 82-day full-slate schedule where every team plays once per league day, simulated non-user games, play-in, best-of-7 playoffs, and boxscores for completed games.

## Play

Open `index.html` in a browser.

## Controls

- `WASD` or arrow keys: Move
- `Space`: Jump ball / hold and release to shoot
- `C`: Dunk near the hoop
- `E`: Pass
- `Q`: Steal on defense
- `F`: Block on defense
- `Shift`: Sprint
- `P`: Pause / resume
- `R`: Restart
